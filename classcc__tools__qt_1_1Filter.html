cc_tools_qt::Filter Class Reference

#include "cc_tools_qt/Filter.h"

Detailed Description

Main polymorphic interface class for filters.

The filter can be used to implement encryption / decryption of the data, and/or additional transport layer.

bool cc_tools_qt::Filter::start ( )

Start filter's operation.

Prior to being started, the filter is not allowed to generate any events. The function invokes virtual startImpl() which can be overriden by the derived class.

Returns
true in case the socket's operation has been successfully started, false otherwise.
